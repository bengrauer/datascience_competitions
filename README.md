# datascience_competitions
This directory houses my data science competitions / scripts I used to compete.  All scripts at this point are created from scratch individually and never forked/copied from other individuals. 

Kaggle Profile: [www.kaggle.com/bengrauer](www.kaggle.com/bengrauer)

## Avito Demand Prediction Challenge (June 2018)
- Sub-Directory: [AvitoDemandPrediction](https://github.com/bengrauer/datascience_competitions/tree/master/AvitoDemandPrediction)
- WebLink: https://www.kaggle.com/c/avito-demand-prediction/
- Final Place: 1,678 out of 1,917
- Code/Model: Python | Random Forest Regressor
- Description: Predict demand for an online classified ad.
- Size: 2MM obs. Images - 70 GB
